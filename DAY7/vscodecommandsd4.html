VS CODE

It is an IDE(Integrated Development Environment).It is an open source code editor which is developed by micrrosoft. It provides tools and resources for developers with a GUI(Graphical User Interface).It is used to write code and debug.

Features Of VS CODE

        1.Open source.
        2.Platform Support.
        3.Development Capabilities.
        4.Integrated Terminal.
        5.Extensions.
        6.Debugging.

Short cuts

Shortcut Description
Ctrl+S Saving the Current file.
Ctrl+C Copying the selected text or item.
Ctrl+V Paste the copied content.
Ctrl+X Cut the selected text or item.
Ctrl+Z Undo the last action.
Ctrl+Y Redo the previously undone action.
Ctrl+A Select all the content in the current file.
Ctrl+Plus(+) zoom in the editor view.
Ctrl+Minus(-) zoom outthe editor view.
Ctrl+Shift+Alt+Down Addcursors to multiple lines for simultaneous editing.
Alt+Up Arrow Move the selected line or block of code one line up.
Alt+Down Arrow Move the selected line or block of code one line down.
Ctrl+Shift+P Open Command Palette
Ctrl+Shift+N New window
Ctrl+Shift+W Close window
